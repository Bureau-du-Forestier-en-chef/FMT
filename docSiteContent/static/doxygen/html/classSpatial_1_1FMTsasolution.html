Detailed Description

This class is a map containing a linear graph for each pixel. The graph represent the action and transition at each each period for the pixel. It's the solution of the FMTsamodel. It's in this class that the "move" is implemented in the function perturb graph. It's also in this class that the value of the objective function for the solution is store.
